# AI README Generator

An AI-powered web application that generates professional README.md files from user input. It helps developers quickly create structured documentation without writing everything manually.

## Features
- AI-powered README generation
- Live preview of generated markdown
- One-click download as `README.md`
- Copy to clipboard functionality
- Retry generation on failure
- Responsive Tailwind UI

## Tech Stack
- React + TypeScript (frontend)
- Node.js + Express (backend)
- Tailwind CSS
- External AI API

## Installation
```bash
# 1. Clone repo
git clone https://github.com/Cemrecicek/ai-readme-generator.git
cd ai-readme-generator

# 2. Install deps
npm install


# 3. Start dev servers
npm run dev          # frontend
npm run server       # backend
```

## Usage
1. Open http://localhost:3000
2. Fill in project name, description, tech stack, and features
3. Click “Generate README”
4. Review the preview pane
5. Hit “Download” to save the file
